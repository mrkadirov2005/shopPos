Shop CRM API Endpoints

Authentication & User Management

  • POST/auth/register — Register new admin or superadmin user
  • POST/auth/login — Login (returns token)
  • POST/auth/logout — Logout user
  • POST/auth/refresh-token — Refresh access token
  • GET/users/:uuid — Get user profile info
  • PUT/users/:uuid — Update user info
  • DELETE/users/:uuid — Delete user
  • GET/users — List all users

Products

  • GET/products — List all products (filters supported)
  • GET/products/:id — Get product details
  • POST/products — Create new product
  • PUT/products/:id — Update product details
  • PATCH/products/:id/restock — Restock product
  • DELETE/products/:id — Delete product

Categories

  • GET/categories — List categories
  • GET/categories/:name — Get category by name
  • POST/categories — Add category
  • PUT/categories/:name — Update category
  • DELETE/categories/:name — Delete category

Brands

  • GET/brands — List brands
  • GET/brands/:name — Get brand details
  • POST/brands — Add brand
  • PUT/brands/:name — Update brand
  • DELETE/brands/:name — Delete brand

Sales

  • GET/sales — List sales (filterable)
  • GET/sales/:sale_id — Get sale details
  • POST/sales — Create new sale
  • PUT/sales/:sale_id — Update sale
  • DELETE/sales/:sale_id — Delete sale

Statistics & Reports

  • GET/stats/week — Weekly sales stats
  • GET/stats/month — Monthly sales stats
  • GET/stats/profit — Profit reports
  • GET/stats/inventory — Inventory levels

Permissions

  • GET/permissions — List permissions
  • POST/permissions — Create permission
  • PUT/permissions/:id — Update permission
  • DELETE/permissions/:id — Delete permission

Admins / Users

  • GET/admins — List admins
  • GET/admins/:uuid — Get admin details
  • POST/admins — Create admin
  • PUT/admins/:uuid — Update admin
  • DELETE/admins/:uuid — Delete admin

Suppliers (Optional)

  • GET/suppliers — List suppliers
  • POST/suppliers — Add supplier
  • PUT/suppliers/:id — Update supplier
  • DELETE/suppliers/:id — Delete supplier
