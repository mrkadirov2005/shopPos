Shop CRM Tech Stack Overview

Backend

  • Node.js
  • Express.js
  • JWT (JSON Web Tokens) for authentication
  • No need for bcrypt (password hashing)
  • Mongoose (MongoDB ODM)
  • Nodemon (development tool for auto restart)

Frontend

  • React.js
  • Tailwind CSS
  • Regular CSS
  • Material-UI (MUI)
  • React Icons

Database

  • MongoDB
  • Mongoose (for schema & querying)

Storage

  • Cloudflare CDN (for storing images and static assets)

Backend Hosting

  • Ahost
  • Render.com
  • Railway

Frontend Hosting

  • Vercel
  • Ahost domain
